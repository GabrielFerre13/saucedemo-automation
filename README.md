#  Test Automation - SauceDemo

Projeto de automação de testes end-to-end simulando o fluxo de compra em um e-commerce.

---

##  Tecnologias utilizadas

- Java
- Selenium WebDriver
- Cucumber (BDD)
- JUnit
- WebDriverManager

---

##  Cenários automatizados

✔️ Login no sistema  
✔️ Adição de produto ao carrinho  
✔️ Início do checkout  

---

## Estrutura do projeto

src/test/java

- pages → Mapeamento dos elementos (Page Object)
- steps → Steps do Cucumber
- interactions → Regras de negócio
- runner → Execução dos testes
- hooks → Setup e teardown

---

##  Aprendizados

- Uso de WebDriverWait para sincronização
- Aplicação de Page Object Model (POM)
- Organização de testes com Cucumber (BDD)
- Separação de responsabilidades (Steps, Pages, Interactions)
